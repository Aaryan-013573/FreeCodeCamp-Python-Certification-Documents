## Module progress (13 modules, in order)

Each module follows the same rhythm: **lessons → workshops → labs → review → 20-question quiz** (18/20 to pass).

- [ ] **1 · Python Basics** — installation, syntax, variables, data types, `print()`
- [ ] **2 · Working with Strings** — slicing, string methods, concatenation, f-strings
- [ ] **3 · Booleans and Conditionals** — truthy/falsy, boolean operators, `if`/`elif`/`else`
- [ ] **4 · Loops and Sequences** — `for` and `while` loops, ranges, iteration
- [ ] **5 · Dictionaries and Sets** — dictionaries, sets, working with modules
- [ ] **6 · Error Handling** — `try`/`except`/`finally`, raising exceptions
- [ ] **7 · Classes and Objects** — class definitions, `__init__`, instances, dunder methods   🎯 *Project #1*
- [ ] **8 · Object-Oriented Programming** — encapsulation, inheritance, polymorphism, abstraction   🎯 *Project #2*
- [ ] **9 · Linear Data Structures** — lists, stacks, queues, linked lists, hash tables   🎯 *Project #3*
- [ ] **10 · Algorithms** — searching and sorting (binary search, merge sort, quicksort, selection sort)   🎯 *Project #4*
- [ ] **11 · Graphs and Trees** — graph/tree concepts, adjacency lists, traversal   🎯 *Project #5*
- [ ] **12 · Dynamic Programming** — memoization, tabulation
- [ ] **13 · Review: Python** — cumulative review module to prepare for the final exam
